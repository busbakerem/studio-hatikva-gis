# סיכום סשן — TLV GIS MCP + מחקר שכונת התקווה
> תאריך: 30.3.2026

## מה נבנה

### שרת MCP (tlv-gis-mcp)
שרת MCP ב-TypeScript עם 10 כלים לשאילתות GIS:
- `query_building_permits`, `query_layer`, `get_building_documents`, `export_map_image`, `get_layer_list`, `identify_at_point` — שרת TLV GIS (IView2 MapServer)
- `export_geojson` — ייצוא GeoJSON עם pagination
- `query_govmap` — קדסטר לאומי (GovMap WFS)
- `search_taba_plans`, `search_datagov` — data.gov.il

מותקן ב-Claude Desktop config. קוד ב-`src/`, build ב-`dist/`.

### ממיר GeoJSON→DXF
`src/geojson-to-dxf.ts` — מבנים תלת-מימדיים (extrusion מגובה) + חלקות דו-מימדיות. עובד עם Rhino.
